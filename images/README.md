# Hero slideshow images

Drop your images here using these exact filenames (any of them can be missing —
the slideshow just skips it):

- `hero-1.jpg`
- `hero-2.jpg`
- `hero-3.jpg`
- `hero-4.jpg`
- `hero-5.jpg`

Recommended: landscape/wide photos (at least 1600×900), since they'll fill the
whole hero background with a dark overlay + white text on top — pick photos
that aren't too busy in the center where the text sits.

`.jpg` or `.png` both work — just match the filename exactly (case-sensitive
on GitHub Pages).
